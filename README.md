# Simpletron 
### 프로젝트 개요
> <b>SML(Simpletron Machine Language)</b> 로 작동되는 <b>기계어 프로그램</b>을 만드는 코딩 과제입니다.</br>
> 컴퓨터를 "완전히 열고" 내부 구조를 살펴봤습니다.

</br>

### 프로젝트 설명
> Simpletron의 모든 명령은 <b>4개의 십진수</b>로 나타내고, 명령이 저장될 메모리 수는 00~99 까지 입니다.
+ operation : 특정 행동을 수행할 연산 코드
+ operand : 메모리 위치

+ SML 의 연산 코드 표
